	\section{Introduction}
		This project will consist in building a virtual Hospital Management System(VHMS). The VHMS will  be
		able to simulate the administrative functions of the hospital and manage them autonomously. The aim
		is to create a system that requires as little human intervention as possible.\\
		And it will have the following features:\\

		\begin{enumerate}
			\item Patient appointment \& Scheduling
			\item Automated Billing
			\item employee tracking
			\item Stock management
		\end{enumerate}
